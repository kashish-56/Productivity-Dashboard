# 🚀 Productivity Dashboard

A modern and responsive **Productivity Dashboard** built using **HTML, CSS, and JavaScript**. This project combines multiple productivity tools into a single dashboard to help users stay organized, focused, and motivated throughout the day.

---

## 🚀 Live Demo

🔗 **View Project:**  
[https://kashish-56/productivity-dashboard/](https://productivity-dashboard-seven-delta.vercel.app/)

---

## ✨ Features

### 📝 Todo List
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Save tasks using Local Storage
- Persistent data after page refresh

---

### 🎯 Daily Goals
- Create daily goals
- Mark goals as completed
- Live progress tracker
- Delete goals
- Automatically saved in Local Storage

---

### ⏳ Pomodoro Timer
- Countdown timer
- Start, Pause, and Reset controls
- Session completion notification
- Prevents multiple timers from running
- MM:SS formatted timer

---

### 💬 Motivational Quotes
- Fetch random motivational quotes using an API
- Display author name
- Loading state while fetching
- Error handling for failed requests
- Refresh quotes with one click

---

### 🌤 Weather Widget
- Displays current weather
- Uses Geolocation API
- Shows:
  - Temperature
  - Weather Condition
  - Humidity
  - Wind Speed
  - Location

---

### 🌙 Dark / Light Mode
- Toggle between dark and light themes
- Theme saved using Local Storage
- Smooth theme transitions

---

### 🎥 Animated Background
- Beautiful fullscreen background video
- Supports dynamic video changes

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Local Storage API
- Fetch API
- Geolocation API
- Font Awesome

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/kashish-56/productivity-dashboard.git
```

### Navigate to the project

```bash
cd productivity-dashboard
```

### Run the project

Simply open **index.html** in your browser.

Or use **VS Code Live Server**.
